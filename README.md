# Kigali Directory

A Flutter mobile app for discovering and sharing locations in Kigali.

## Features

- User authentication (Sign up, Login, Email verification)
- Create, edit, and delete location listings
- Search listings by name
- Filter listings by category
- Interactive map view with markers
- Get directions to locations

## Tech Stack

- Flutter & Dart
- Firebase Authentication
- Cloud Firestore
- OpenStreetMap (flutter_map)
- Riverpod (State Management)

## Setup

1. Clone the repository
2. Run `flutter pub get`
3. Configure Firebase (add `google-services.json` for Android)
4. Run `flutter run`
